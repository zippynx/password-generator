# 🔐 Password Generator

Aplikasi web sederhana untuk membuat password secara acak menggunakan HTML, CSS, dan JavaScript.

## ✨ Fitur

* Generate password dengan panjang 8–32 karakter
* Pilihan karakter:

  * Huruf kecil (a-z)
  * Huruf besar (A-Z)
  * Angka (0-9)
  * Simbol (!@#$%^&*-+=?)
* Tombol copy password ke clipboard
* Tampilan sederhana dan mudah digunakan

## 🛠️ Teknologi

* HTML5
* CSS3
* JavaScript

## 📂 Struktur File

```text
password-generator/
├── index.html
├── style.css
├── password_generator.js
└── README.md
```

## 🚀 Cara Menjalankan

1. Download atau clone repository ini.
2. Buka file `index.html` menggunakan browser.
3. Pilih jenis karakter yang diinginkan.
4. Atur panjang password menggunakan slider.
5. Klik tombol **Generate**.
6. Klik **Copy** untuk menyalin password.

## 📌 Deskripsi

Project ini dibuat untuk mempelajari penggunaan JavaScript dalam menghasilkan password acak, manipulasi DOM, serta event handling pada halaman web.
